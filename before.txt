13.5 繰り込み群

第13.4節では、臨界指数の値を決定するために、異なる長さのスケールでさまざまな量の特性を調べた。異なる長さのスケールで臨界点近傍の物理量を調べるという考え方は、有限サイズ・スケーリングを超えて拡張され、ここ25年ほどの間に理論物理学で発展したおそらく最も重要な新しい方法である繰り込み群の基礎になっている。ウィルソン(Kenneth Wilson)は、くりこみ群の方法の発展への寄与により、1981年にノーベル物理学賞を受賞した。その方法は初め熱力学相転移に適用されたが、パーコレーション転移を題材として導入するほうがより簡明である。繰り込み群の方法で臨界指数を直接求めることができること、そして、モンテカルロ法と組み合わせると、多くの場合モンテカルロ法だけよりさらに強力になることが以下で分かるだろう。

繰り込み群の方法を導入するために、$p=p_{0} < p_{c}$で生成したパーコレーションの配置を撮った写真を考えてみる。だんだんと遠く離れてその写真(またはスクリーン)を見るとどう見えるだろうか。写真からずっと離れたときには、互いに隣り合った占有された格子点を見分けることはできないし、単独の格子点からなるクラスターは見えなくなってしまう。さらに、大きいクラスターから出た枝や大きい”塊”どうしを連結する狭い橋も、遠くからでは見えなくなってしまう。したがって、$p_{0} < p_{c}$では、遠くからみた写真はあたかも$p_{0}$よりも小さい値$p=p_{1}$で生成されたパーコレーションの配置のように見える。また、まだ見えているクラスターの結合長$\xi(p_{i})$は$\xi(p_{0})$より小さい。写真からさらに離れると、見えるクラスターは、$p_{1}$より小さい$p=p_{2}$のときの、より小さいクラスターのように見える。最終的には、どのクラスターも識別できなくなって、写真は自明な固定点$p=0$にあるかのように見えるだろう。

$p_{0}> p_{c}$の場合には、離れるにつれて写真はどう見えるだろうか。上と同じ理由から、占有されていない格子点の小さい領域だけが見えるようになると推測できる。写真からさらに離れると、これらの領域はさらに見えにくくなって、格子のより大きい割合が占有されている配置に見える。つまり、写真は$p_{0}$より大きい$p=p_{1}$の値で生成された、$\xi(p_{1}) < \xi(p_{0})$であるような配置に見える。写真からさらに遠ざかれば、結局、もう1つの自明な固定点$p=1$にあるかのように見えるようになる。

それでは、$p_{0} = p_{c}$ではどうなるだろうか。パーコレーションのしきい値では、すべての長さのスケールが存在し、系を観測するためにどの長さのスケールを使うかは問題でなくなる。つまり、写真を見る距離に関わらず、(全体的には小さくなるが)写真は同じように見える。この意味で、$p_{c}$は特別な自明でない固定点である。

そこで、写真から遠ざかるのと同様のことを、コンピュータを使って、配置を変える方法で考える。格子全体を覆うセルあるいはブロックで分割された正方格子を考える。セル内のすべての格子点が融合して新しい1個の超格子点、すなわち繰り込まれた格子点になるという見方で格子を見るなら、その新しい格子はもとの格子と同じ対称性を持つ。しかし、セルを新しい格子点で置き換えることで長さのスケールが変化する。セルの1辺の長さを$b$とすると、すべての長さは因子$b$だけ小さくなる。したがって、繰り込みの効果は、各格子点のグループを1つの繰り込まれた格子点に置き換え、繰り込まれた格子の結合長を因子$b$でスケールし直すことである。
繰り込まれた格子点が占有されているかどうかは、どのように決めればよいだろうか。もとの格子の主な特徴と、したがってその結合性(と対称性)も保持したいので、もとの格子点の集まりがそのセルの端から端まで連結したときに、繰り込まれた格子点は占有されるとする。便宜上橋から端まで縦に連結する判定条件を採用しよう。図13.11と図13.12には、$p$が$p_{c}$の上と下のとｋの典型的なパーコレーションの配置に対する繰り込み群変換が示されている。どちらの場合もさらに変換を続けて行うと、系は$p_{c}$から離れていく。$p=0.7$では変換の結果、系は$p=1$に向かって動き、$p=0.5$では$p=0$に向かう。以下で議論するが、$p_{c}$は繰り込み群変換の不安定固定点に関係づけることができる。もちろん有限な大きさの格子で変換を始めたので、繰り込み群変換を無限に続けることはできない。

視覚による繰り込み群でも$p_{c}$を求めることはできるが、臨界指数はもｔまら内。以下に$p_{c}$と結合長の臨界指数を求めることができる繰り込み群の方法を示す。この解析法はレイノルズ(Reynolds)等の与えた方法に忠実に従うものである(参考文献)。

彼らの繰り込み群の方法は、(i)繰り込まれた配置の状態を定めるすべての変数を指定して、基礎的な変数について平均をとる,(ii)繰り込まれた配置を、もとのパラメータとそれ以外のパラメータで表す、という2つの部分からなっている。以前と同じ平均法を採用することにして、1辺の長さ$b$のセル内の$b^{d}$個の格子点を、もとの格子がそのセルの端から端を連結しているかどうかを表す1個の格子点で置き換える。次の手順は、どのパラメータで平均後の新しい配置を指定するかを決めることである。ここで、各セルが他のすべてのセルとは無関係に、そのセルが占有される確率$p'$を持つという簡単な近似を行うことにする。セルは端から端まで連結した格子点の組を含むときにのみ占有されると定めているので、$p$、$p'$の繰り込み群変換は、パーコレーションの基本的な物理が結合の仕方であることを反映している。格子点が確率$p$で占有される場合に、セルは

p' = R(p)

の形の繰り込み変換、つまり漸化式で与えられる。量$R(p)$は、格子点が端から端まで連結した経路を形成する確率である。

式(13.18)は形式的なので、例を考えたほうがわかりやすいだろう。図13.13に、$b=2$のセルの端から端を立てに連結する7個の配置が示されている。繰り込まれた格子点が占有される確率$p'$は、端から端まで連結したすべての配置の確率の和

p' = R(p) = p^{4} + 4p^{3} (1-p) + 2p^{2}(1-p)^{2}

で与えられる。一般的に、繰り込まれた格子点が占有される確率$p'$は、もとの格子点の占有される確率$p$とは異なる。たとえば、$p = p_{0} = 0.5$から始めるとする。1回めのくりこみ変換の後には、式(13.19)から得られる$p'$の値は$p_{1} = p' = R(p_{0} = 0.5) = 0.44$である。2回めの繰り込み変換を行うと、$p_{2} = R(p_{1} = 0.35)$が得られる。さらに変換を続けると、系は固定点$p=0$に向かう。同様にしてp= p_{0}=0.7$から始めた場合、変換を続けると、系は固定点$p=1$に向かうことがわかる。この振る舞いは、視覚的に繰り込み群操作を行ったときと、定性的に似ている。

しきい値$p_{c}$と関係した、自明でない固定点を見つけるには、

p^{*} = R(p^{*})

を満たす特別な$p$の値を求めることが必要である。漸化式(13.19)については、$p^{*}$の4次方程式の下位として、2個の自明な固定点$p^{*} = 0$と$p^{*} = 1$の他に、$p_{c}$と関係した自明でない固定点$p^{*} = 0.61804$が得られる。この$b=2$の場合の$p^{*}$の計算値は、前に得られた値$p_{c} = 0.5927$と比較すべき値である。

臨界指数$\nu$を計算するには、繰り込まれた格子では、もとの系の長さに対して、すべての長さが係数$b$だけ縮小されていることを思い起こす必要がある。したがって結合長は

\xi' = \xi /b

で変換される。$p \sim p_{c}$では$\xi(p) = const|p-p_{c}|^{-\nu}$なので

|p' - p^{*}|^{-\nu} = b^{-1} |p - p^{*}|^{-\nu}

ただし$p_{c}$を$p^{*}$に等しいとした。$p_{c}$近傍で$p'$と$p$の関係を得るために、繰り込み変換(13.18)を$p^{*}$のまわりでテイラー級数に展開すると、(p-p^{*})の1次までで

p' - p^{*} = R(p) - R(p^{*} \approx \lambda (p-p^{*}))

が得られる。ここで

\lambda = \frac{dR(p = p^{*})}{dp}

である。$\nu$の式を導くには少しばかりの代数計算が必要である。まず、式(13.23)の両辺を$\nu$乗し、

|p' -p^{*}|^{\nu} = \lambda|p - p^{*}|^{\nu}

と書く。そして、式(13.25)と(13.22)を比較すると

b = \lambda ^{\nu}

を得る。最後に式(13.26)の両辺の対数をとると、求めていた臨界指数$\nu$の関係式

\nu = \frac{\log b}{\log \lambda}

が得られる。

例として、正方格子で$b=2$としたときの$\lambda$を計算しよう。式(13.19)を$R(p) = -p^{4} + 2p^{2}$の形に書く。$R(p)$を$p$で微分すると、$p=p^{*}=0.61804$では、$\lambda = 4p(1-p^{2}) = 1.5279$が得られる。そこで式(13.27)を使うと、

\nu = \frac{\log 2}{\log 1.5729} = 1.635 \cdots

を得る。値(13.28)を2次元の正確な値$\nu = 4/3$(表13.1参照)と比較すると、このような簡単計算でも、かなりよい一致が得られることが分かる。($2\times 2$の格子で直接$\xi(p)$を求めると、どのような結論になるだろうか。)しかし$\nu$の計算の精度は分からない。ここで使った近似の特徴は何だろうか。主な仮定は、各セルが占有されるかどうかは他のすべてのセルとは無関係に決まるとしたことだった。この仮定はもとの格子については正しいが、1回の繰り込み変換でもとの格子の連結した経路の一部は失われ、、もとの格子点にはない連結した経路が現れる。この境界面で生じる問題の例が図13.14に示されている。この境界面効果はセルのサイズを大きくするとともに確率的に小さくなるので、繰り込み群の計算を改善する1つの方法は、より大きいセルわ考えることである。問題13.11dでは$b=3$の計算を考える。課題13.2では、繰り込み群の方法をモンテカルロ法と結びつけて、さらに大きいセルを取り扱う。



問題14.2 フラクタル次元の繰り込み群による計算

もとの格子と、大きさ$L' = L/b$に繰り込まれた格子の両方で、$p=p_{c}$における端から端まで連結したクラスター内の占有された格子点の数の2乗$<M^{2}>$および、$<M'^{2}>$をそれぞれ計算せよ。$<M^{2} \sim R^{2D}>$であり、$<M'^{2} \sim (R/d)^{2D}>$でもあるので、関係式$b^{2D} = <M^{2}>/<M'^{2}>$から$D$を求めることができる。長さのスケール変換のための因子を$b=2$と選んで、第113.5節で用いたのと同じブロック法を用いよ。$L=16$、$p=0.5927$の場合に定性的な結果を得るには、10個の端から端まで連結したクラスターについて平均を取れば十分である。
